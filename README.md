# Streamlit Blog App

This is a simple Streamlit app that allows users to write and publish blogs. Users can input an API key (e.g., Cohere) to enhance blogging functionality.

## Features
- Input API key for enhanced features
- Write and publish blogs
- Simple and intuitive interface

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## Usage
1. Enter your API key in the sidebar.
2. Write your blog title and content.
3. Click "Publish Blog" to save your blog.
